# 设备服务与工单协同平台 Beta

微信原生小程序骨架，覆盖以下 Beta 主链路：

- 登录与角色切换
- 多租户与 RBAC 视图控制
- 首页工作台、扫码、消息提醒
- 工单创建、处理、完结
- 设备列表、状态监控、详情与报警
- 发现页内容分发
- 我的页面、员工管理、设备分配、设置

## 目录

- `app.*`: 全局配置与主题
- `pages/`: 页面实现
- `utils/mock.js`: 本地模拟数据
- `utils/services.js`: 面向真实接口替换的服务层
- `utils/session.js`: 当前登录态与偏好存储
- `utils/permissions.js`: 菜单、按钮、数据可见范围控制

## 本地运行

1. 使用微信开发者工具打开当前目录。
2. 选择“小程序”项目，`appid` 可先用体验号或测试号替换。
3. 默认会自动登录成内部工程师，可在“我的”页面切换演示身份。

## 接口替换建议

当前所有页面都经由 `utils/services.js` 取数。接入真实后台时，建议保留服务层签名不变，仅将 mock 数据替换为 `wx.request` 或内部 SDK。
