## 功能特性

### 核心功能
- ✅ **多角色登录** - 支持工程师、租户管理员、客户员工三种角色
- ✅ **多租户管理** - 基于 RBAC 的数据隔离与权限控制
- ✅ **首页工作台** - 快速访问工单、设备、消息、知识库
- ✅ **工单管理** - 创建、接单、处理、完结完整流程
- ✅ **设备监控** - 实时状态、参数查看、告警通知
- ✅ **物料管理** - 设备系列、型号、物料库存管理
- ✅ **扫码功能** - 快速访问设备和工单详情
- ✅ **多语言支持** - 中文/英文双语界面

### 技术特性
- ✅ **真实数据对接** - 已集成 K-Cloud 后端 API
- ✅ **智能降级机制** - API 失败时自动降级到 mock 数据
- ✅ **完整的错误处理** - 网络异常、Token 过期等友好提示
- ✅ **数据映射层** - 自动处理前后端数据格式差异

- 微信开发者工具（最新稳定版）
- 微信小程序开发权限（AppID）

### 运行步骤

1. **打开项目**

2. **关闭域名校验**（开发阶段）

   - 首次打开会自动登录为内部工程师
   - 可在”我的”页面切换演示身份

### 数据映射说明
| 模块 | 后端字段 | 前端字段 | 映射规则 |
|------|---------|---------|---------|
| 设备 | `id` | `deviceId` | 数字转字符串 |
| 设备 | `telemetries` | `params` | 遥测数据转参数列表 |
| 工单 | `ticket_no` | `workOrderId` | 字段名映射 |
| 工单 | `type` | `type` | 类型翻译（fault→维修工单） |
| 工单 | `comments` | `logs` | 评论转日志格式 |
| 物料 | `sku` | `spec` | 字段名映射 |
| 物料 | `stock_quantity` | `stockQuantity` | 下划线转驼峰 |

## 开发指南

### 添加新页面
1. 在 `pages/` 目录创建新页面
2. 在 `app.json` 的 `pages` 数组中注册
3. 在 `utils/services.js` 添加对应的数据服务

### 修改样式
- 全局样式：`app.wxss`
- 主题色：`utils/constants.js`

### 添加新的 API 接口
1. 在 `utils/request.js` 中已封装通用请求方法
2. 在 `utils/services.js` 中添加业务函数
3. 在 `utils/mappers.js` 中添加数据映射（如需要）

## 生产环境部署

### 必须完成的配置
1. **HTTPS 证书**
   - 微信小程序要求使用 HTTPS
   - 后端需要配置有效的 SSL 证书

2. **域名白名单**
   - 在微信公众平台配置合法域名
   - 后端需要配置 CORS 允许小程序域名

3. **AppID 配置**
   - 在 `app.json` 中使用正式的小程序 AppID

4. **移除降级逻辑**
   - 移除 mock 数据降级代码
   - 确保所有 API 都能正常访问
